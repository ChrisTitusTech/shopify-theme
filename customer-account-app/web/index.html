Order Downloads

customer-account.order-status.cart-line-list.render-after
New Customer Accounts — Order Detail page (after line items)
At least one line item has a non-empty custom.url metafield

Metafield Setup Checklist

  1. Metafield exists: Products → Custom data → namespace custom, key url, type list.url
  2. Customer Account API access enabled:
    Shopify Admin → Settings → Custom data → Products → custom.url → Edit → enable "Customer Account API" read access.
    ⚠ Without this the GraphQL query returns null and no download buttons appear.
  3. Product has a value: Open any product, scroll to Metafields, confirm custom.url has at least one URL entry.

GraphQL Query (Customer Account API)

The extension runs this query client-side after the order loads: 

query OrderDownloads($orderId: ID!) {
  order(id: $orderId) {
    lineItems(first: 50) {
      nodes {
        title
        product {
          metafield(namespace: "custom", key: "url") {
            value
          }
        }
      }
    }
  }
}

Returns JSON-encoded list.url array; the extension takes the last entry.
